# ELD Log Generator - Frontend

React frontend for the ELD Log Generator application that helps commercial truck drivers create FMCSA-compliant electronic logging device records.

## Features

- **Trip Planning**: Calculate routes and driving schedules
- **HOS Compliance**: Automatic Hours of Service calculations
- **ELD Log Generation**: Create compliant daily log sheets
- **Interactive Maps**: Visualize routes with OpenStreetMap
- **Printable Logs**: Generate PDF-ready ELD logs
- **Multiple Day Support**: Handle trips spanning multiple days

## Quick Start

1. **Install dependencies:**
   ```bash
   npm install
   ```
